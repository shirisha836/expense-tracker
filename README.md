# 💰 Expense Tracker

A responsive web application designed to help users efficiently track and manage their daily expenses with a simple and intuitive interface.

---

## 🚀 Features

* Add and delete expenses easily
* Real-time calculation of total expenses
* Clean and user-friendly interface
* Lightweight and fast performance

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 📌 Project Overview

Managing daily expenses is essential for financial awareness. This project helps users maintain a clear record of their spending habits and provides a simple way to monitor expenses without complexity.

---

## 📂 Folder Structure

```
expense-tracker/
│── index.html
│── style.css
│── script.js
```

---

## ⚙️ How to Run

1. Clone the repository
2. Open the project folder
3. Double-click `index.html` to run in browser
